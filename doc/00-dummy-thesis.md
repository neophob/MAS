# Einleitung

Dieses Dummy-Dokument prueft den gesamten Workflow von Markdown ueber HTML bis PDF.

![Vorlagenbild aus dem Template](../mdgen/.cache/template-assets/image4.jpeg)

## Zielsetzung

Der Generator soll alle Markdown-Dateien aus dem `doc`-Verzeichnis einsammeln, Metadaten uebernehmen und ein druckfaehiges Thesis-PDF erzeugen.

# Methodik

Die Pipeline besteht aus drei Schritten:

1. Markdown-Dateien werden sortiert und zusammengefuehrt.
2. Das Layout wird mit HTML und CSS an die Thesisvorlage angelehnt.
3. Ein Headless-Browser rendert das Ergebnis in eine PDF-Datei.
