# Einleitung

Ersetze dieses Starter-Dokument durch den echten Bericht. Alle Markdown-Dateien im `doc/`-Verzeichnis werden in alphabetischer Reihenfolge zusammengefuehrt.

![Et ut aut isti repuditis qui ium](../mdgen/.cache/template-assets/image4.jpeg)

## Zielsetzung

Der Generator sammelt die Kapiteldateien aus dem `doc`-Verzeichnis ein, uebernimmt die Metadaten aus der Datei mit `role: "title"`, erzeugt das Inhaltsverzeichnis automatisch und rendert ein druckfaehiges Thesis-PDF.

# Methodik

Die Pipeline besteht aus drei Schritten:

1. Markdown-Dateien werden sortiert und zusammengefuehrt.
2. Das Layout wird mit HTML und CSS an die Thesisvorlage angelehnt.
3. Ein Headless-Browser rendert das Ergebnis in eine PDF-Datei.
